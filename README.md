# notes-site

React to-do list frontend that connects to a backend API for persistent note management.

## Context
- **Course:** Ksquare University Bootcamp
- **Date:** 2021

## Tech Stack
- React 17
- Create React App
- Fetch API

## What It Does
A "To-doitealo" notes app with a UI for creating, viewing, and deleting to-do items. Uses React hooks (useState, useEffect) to fetch notes from a backend API. Users can add new notes with a title and content, and delete individual items. Designed as the frontend companion to the Web_Service_API project.
